## Como Adicionar

1. Baixar/converter imagens para WebP
2. Colocar em `public/images/depo/` (influencers) ou `public/images/` (CTA)
3. Nomes devem ser exatamente como especificado acima
4. Se usar placeholders temporários, o site vai mostrar "imagem não disponível" até adicionar as reais
